# EnOcean Module for M5Stack Unit

EnOceanセンサーをM5Stackで扱うためのGrove接続モジュールで

---

## 特徴

- Grove接続で簡単に利用可能
- EnOcean無線の受信・送信に対応
- M5Stackシリーズに最適化

---

## 必要なもの

- M5Stack本体
- 本モジュール（enocean-m5-unit）
- EnOcean対応センサー

---

## 接続方法

Groveポートに接続するだけで利用できます。

---

## 想定用途

- 温度・湿度センサ
- ドア開閉検知
- 無線スイッチ入力
- IoTデータ取得
